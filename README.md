# Cumulus
Unobtrusive SoundCloud desktop application

# Developing

## Install dependencies
`npm install`

`npm install -g electron-prebuilt`

## Compile the application
`grunt` or `grunt build`

## Run the application
`NODE_ENV=development electron .`
